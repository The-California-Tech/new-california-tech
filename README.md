# The California Tech

A SvelteKit-powered site for The California Tech, using Notion/Supabase sync via Symbiont and QWER content tooling.

## Development

Prerequisites:
- Node.js 18+
- pnpm

Install:

```bash
pnpm install
```

Run locally:

```bash
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Deployment

This repository is configured for deployment on Vercel with `@sveltejs/adapter-vercel`.
